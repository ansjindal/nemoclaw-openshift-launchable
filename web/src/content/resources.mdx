Where to go deeper, and the key links.

## The stack
- **Red Hat OpenShift** — docs.openshift.com
- **MicroShift** — microshift.io (the OKD build & MINC)
- **NVIDIA OpenShell** — the agent sandbox runtime & control plane
- **OpenClaw** — the agent & its control UI; `openclaw --help`, docs.openclaw.ai
- **NVIDIA Brev** — brev.nvidia.com (instances & credits)
- **build.nvidia.com** — OpenAI-compatible model endpoints (NIM)

## Concepts worth a read
- OpenShift RBAC, Security Context Constraints, NetworkPolicy.
- The agent-sandbox CRD (`sandboxes.agents.x-k8s.io`).
- OpenClaw skills, tools, device pairing, and the workspace-as-memory pattern (IDENTITY/SOUL).
- Model Context Protocol (MCP) for typed agent tools.

## The launchable repo
Everything you deployed is scripted in the launchable repo — `scripts/` (the phases), `manifests/` (OpenShell, OpenClaw, console), `helm/`, and this site under `web/`. Read it, fork it, build your hackathon project on top.
